# 🔄 반품 관리

> 💡 **반품/교환 접수 시 히로인스 운영 담당자가 직접 연락 드려요!**

---

## 반품 상태

| 상태 | 설명 |
|------|------|
| 반품요청 | 고객이 반품 요청 |
| 반품완료 | 환불 처리 완료 |

---

## 반품 처리 절차

1. 고객이 히로인스 문의로 반품/교환 접수
2. 히로인스 담당자가 반품 사유 검토
3. 셀러에게 내용 전달
4. 상품 회수
5. 환불 처리

---

## 반품 사유별 배송비

| 사유 | 배송비 부담 |
|------|------|
| 단순 변심 | 고객 |
| 상품 불량 | 판매자 |
| 오배송 | 판매자 |
| 상품 파손 | 배송사 또는 판매자 |

---

> ⚠️ 반품 요청 후 일정 기간 내 미처리 시 **자동 승인**될 수 있어요. 주기적으로 확인해주세요!
