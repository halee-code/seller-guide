# ✍️ 회원가입

간편하게 가입하고 판매를 시작하세요!

***

## 📍 접속하기

👉 [https://seller.heroines.me/](https://seller.heroines.me/)에서 **회원가입** 클릭

***

## 📝 가입 절차

### 1️⃣ 기본 정보 입력

| 항목       | 입력 내용               |
| -------- | ------------------- |
| 아이디(이메일) | 로그인용 이메일            |
| 비밀번호     | 영문+숫자+특수문자 포함 8-16자 |

> 💡 **팁**: 실제 업무용 이메일을 사용하세요! 주문/정산 알림이 발송돼요.

**이메일 인증 방법**

1. 이메일 입력 → **중복 확인**
2. **인증번호 받기** 클릭
3. 이메일로 온 인증번호 입력

***

### 2️⃣ 담당자 정보 입력

| 항목      | 입력 내용  |
| ------- | ------ |
| 담당자 이름  | 본인 이름  |
| 담당자 연락처 | 휴대폰 번호 |

> ⚠️ 가입하시는 본인 정보를 입력해주세요.

***

### 3️⃣ 약관 동의 & 가입 완료

* ✅ 서비스 이용약관 (필수)
* ✅ 개인정보처리방침 (필수)

**회원가입** 버튼 클릭하면 끝!

***

## ➡️ 다음 단계

가입 후 아래 설정을 완료해야 판매할 수 있어요.

1. [판매자 정보 설정](seller-profile.md) → 사업자 정보 등록
2. [배송 정보 설정](../products/shipping-settings.md) → 배송비/반품 정책
