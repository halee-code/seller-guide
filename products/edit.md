# ✏️ 상품 수정

등록된 상품 정보를 수정해보세요!

---

## 📍 접속 경로

**상품관리** → **내 상품** → 수정할 상품 클릭

---

## 수정 방법

1. 상품 목록에서 수정할 상품 클릭
2. 정보 수정
3. **저장** 클릭

---

## 수정 가능 항목

| 항목 | 설명 |
|------|------|
| 상품명 | 이름 변경 |
| 카테고리 | 카테고리 변경 |
| 가격 | 원가, 할인가, 판매가 |
| 재고 | 수량 수정 |
| 이미지 | 대표/상세 이미지 |
| 옵션 | 추가/삭제/수정 |

---

> ⚠️ 주문 진행 중인 상품의 가격/옵션 변경은 **새 주문부터** 적용돼요!
